Regular expressions
-------------------

* Invented as a way to describe strings.
* Popular for all sorts of applications.
* Way too many variants of the syntax (and capabilities)
* Basic concepts:
    * Exact matches
    * Sets
    * Special symbols
    * Kleene *
    * Alternation
    * Parenthesization
* We will generally play with things using `/PATTERN/.match(STRING)`
